# pos_system




A new Flutter project.

## Getting Started

A fully developed Point of Sale (POS) mobile application built using Flutter, designed to provide all the functionalities of a standard POS system. The app includes features such as receipt generation, sales tracking, and team collaboration.

It also integrates time-in and time-out attendance management using the Google Location API, ensuring accurate staff tracking based on location. Additionally, the system supports K40 hardware integration for seamless operational performance.

The application allows cashiers to process sales and monitor their performance on a daily, weekly, and monthly basis. It incorporates a structured role-based access system, where the Administrator holds full control and superuser privileges, followed by Supervisors and Managers, and finally Employees (including cashiers), each with appropriate levels of access and permissions.
