Pixel POS – iOS build (IPA)
==========================

iOS cannot be built on Windows. You need:

  • macOS
  • Xcode
  • Apple Developer account (signing)

Release version (match pubspec): 1.0.3 (build 4)

  1. Copy the whole POS project to a Mac (or clone).
  2. From project root:

       flutter pub get
       flutter build ipa --build-name=1.0.3 --build-number=3

  3. In Xcode (ios/Runner.xcworkspace): Runner target → Signing & Capabilities → your Team.

  4. After a successful build, find the IPA under build/ios/ipa/

  5. Copy it to:

       final app/Pixel_POS_ios_1.0.3_build4.ipa

Or run (from project root on Mac):

  chmod +x "final app/build_ios.sh"
  "./final app/build_ios.sh"

That script runs tests, builds the IPA, removes old Pixel_POS_ios_*.ipa in final app, and copies the new file.
